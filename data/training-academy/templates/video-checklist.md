# Video Production Checklist
## YPP Instructor Academy Training Videos

Use this checklist to ensure every video meets quality standards before publishing.

---

## PRE-PRODUCTION

### Planning
- [ ] Learning objectives defined (specific, observable, actionable)
- [ ] Target duration determined (realistic for content)
- [ ] Module/video title finalized
- [ ] Audience clearly identified (new instructors, experienced, etc.)

### Scripting
- [ ] Script written using template
- [ ] Hook grabs attention in first 30 seconds
- [ ] Content flows logically from simple to complex
- [ ] Examples are specific, diverse, and relatable
- [ ] Language is inclusive and accessible
- [ ] Jargon defined when used
- [ ] Summary recaps key takeaways
- [ ] Clear call-to-action included
- [ ] Script reviewed for tone (supportive, professional, empowering)
- [ ] Script reviewed for length (matches target duration)

### Resources Preparation
- [ ] All templates/downloads mentioned are ready and accessible
- [ ] Links to resources tested and working
- [ ] Resource descriptions accurate
- [ ] Permissions secured for any third-party content

---

## RECORDING DAY

### Environment Setup
- [ ] Quiet location selected (no background noise)
- [ ] Lighting tested (face well-lit, no harsh shadows)
- [ ] Background clean and professional
- [ ] Notifications silenced (phone, computer)
- [ ] Unnecessary apps/windows closed

### Equipment Check
- [ ] Microphone connected and tested
- [ ] Audio levels checked (not too loud/quiet)
- [ ] Camera positioned at eye level
- [ ] Camera focus and framing correct
- [ ] Screen recording software ready (if needed)
- [ ] Second monitor or tablet for script/notes (optional)

### Test Recording
- [ ] 30-second test recording completed
- [ ] Audio quality verified (clear, no background noise)
- [ ] Video quality verified (sharp, well-lit)
- [ ] Framing checked (head and shoulders visible)
- [ ] Test playback reviewed

### Recording
- [ ] Energy level appropriate (enthusiastic but not over-the-top)
- [ ] Speaking clearly and at moderate pace
- [ ] Looking at camera (not screen) when talking head
- [ ] Using script as guide, not reading robotically
- [ ] Pausing 3 seconds before/after each segment (for editing)
- [ ] Marking mistakes (clap or verbal marker) for easy editing
- [ ] Recording in chunks (5-10 min segments max)

### Screen Recording (if applicable)
- [ ] High contrast cursor (large and visible)
- [ ] Mouse movements slow and deliberate
- [ ] Zooming in on important areas
- [ ] Highlighting clicks/actions
- [ ] Minimizing distractions on screen
- [ ] Closing irrelevant tabs/windows

---

## POST-PRODUCTION

### Editing - Rough Cut
- [ ] All footage imported into editing software
- [ ] Mistakes and long pauses removed
- [ ] Filler words cut ("um," "uh," "like")
- [ ] Segments reordered if needed
- [ ] Total duration within target (+/- 10%)

### Editing - Graphics & Visual Elements
- [ ] Title card added (module name, video title)
- [ ] Lower third added (your name and role, if applicable)
- [ ] Key terms/concepts highlighted with text overlays
- [ ] Arrows/callouts added to point to important elements
- [ ] B-roll or supplementary visuals inserted (if applicable)
- [ ] Chapter markers added (for videos >10 min)
- [ ] Transitions are smooth (not distracting)
- [ ] Branding consistent (YPP colors, fonts)

### Editing - Audio
- [ ] Background noise removed/reduced
- [ ] Audio levels normalized (consistent volume)
- [ ] Clipping or distortion fixed
- [ ] Background music added (if appropriate, at 10-20% volume)
- [ ] Music is royalty-free
- [ ] Audio synced properly with video

### Editing - Color Correction
- [ ] Brightness adjusted (not too dark/bright)
- [ ] Contrast appropriate
- [ ] Color saturation natural (not over-saturated)
- [ ] Consistent look throughout video

### Review - First Pass
- [ ] Watch entire video start to finish (no skipping)
- [ ] Check for awkward cuts or jumps
- [ ] Verify on-screen text is readable (test on mobile)
- [ ] Confirm graphics/callouts are accurate
- [ ] Listen for audio issues (clicks, pops, inconsistent volume)
- [ ] Note timestamp of any issues for revision

### Review - Content Check
- [ ] Learning objectives clearly addressed
- [ ] All key points covered
- [ ] Examples are clear and appropriate
- [ ] No factual errors or typos
- [ ] Call-to-action is specific and actionable
- [ ] Tone aligns with YPP voice (supportive, professional, empowering)
- [ ] Length is appropriate (not too long/short)

---

## ACCESSIBILITY

### Closed Captions
- [ ] Captions generated (auto or manual)
- [ ] Captions reviewed for accuracy (100% accuracy required)
- [ ] Captions synced properly with speech
- [ ] Captions formatted correctly (1-2 lines, 32-42 chars/line)
- [ ] Sound effects noted in brackets [music], [applause]
- [ ] Speaker labels included (if multiple speakers)
- [ ] Proper punctuation and capitalization

### Visual Accessibility
- [ ] High contrast between text and background
- [ ] Text size large enough to read on mobile (24pt minimum)
- [ ] Color not the only way to convey information
- [ ] Visual elements described verbally in narration

### Cognitive Accessibility
- [ ] Pacing appropriate (not too fast)
- [ ] Pauses between major concepts
- [ ] Signposting used ("First... Second... Finally...")
- [ ] Summaries provided at logical breakpoints
- [ ] Transcript available (full text version)

---

## EXPORT & UPLOAD

### Export Settings
- [ ] Resolution: 1920x1080 (1080p) or 1280x720 (720p)
- [ ] Frame rate: 24fps or 30fps
- [ ] Format: MP4 (H.264 codec)
- [ ] Bitrate: 5-10 Mbps (1080p) or 2.5-5 Mbps (720p)
- [ ] Audio: AAC, 128-192 kbps, 44.1 kHz
- [ ] File size reasonable (< 500MB for videos under 20 min)

### Final Quality Check
- [ ] Watch exported video in full (before uploading)
- [ ] Test on different devices (desktop, mobile, tablet)
- [ ] Verify audio/video quality maintained after export
- [ ] Check file plays correctly (no corruption)

### Upload to Platform
- [ ] Video title clear and descriptive
- [ ] Description includes:
  - [ ] Learning objectives
  - [ ] Timestamps/chapters (for longer videos)
  - [ ] Links to resources mentioned
  - [ ] Link to training portal
- [ ] Tags/keywords added (for searchability)
- [ ] Custom thumbnail created and uploaded (if applicable)
- [ ] Privacy settings configured (public/unlisted/private)
- [ ] Video added to appropriate playlist
- [ ] Closed captions file uploaded (if not using platform auto-captions)

---

## POST-PUBLISH

### Content Integration
- [ ] Video URL added to content.v2.json
- [ ] Video provider specified (YouTube/Vimeo/Loom)
- [ ] Duration accurately recorded
- [ ] Segments/chapters defined (if applicable)
- [ ] Video marked as primary or supplementary
- [ ] Sort order correct

### Communication
- [ ] YPP Training Team notified of new video
- [ ] Video added to instructor academy platform
- [ ] Instructors notified (if updating existing module)

### Monitoring
- [ ] Analytics dashboard checked (after 1 week)
- [ ] View duration reviewed (are people watching all the way through?)
- [ ] Drop-off points identified (where do viewers stop?)
- [ ] Comments/questions monitored
- [ ] Quiz scores reviewed (are learning objectives being met?)

---

## QUALITY ASSURANCE (Peer Review - Optional but Recommended)

Have a colleague watch the video and assess:

### Content
- [ ] Learning objectives clearly addressed
- [ ] Content accurate and up-to-date
- [ ] Examples relevant and appropriate
- [ ] Tone supportive and professional
- [ ] Clear and actionable takeaways

### Engagement
- [ ] Maintains interest throughout
- [ ] Pacing appropriate
- [ ] Energy level engaging
- [ ] Visual elements enhance (not distract)

### Clarity
- [ ] Explanations easy to understand
- [ ] No confusing jargon or concepts
- [ ] Transitions smooth
- [ ] Call-to-action clear

### Technical Quality
- [ ] Audio clear and consistent
- [ ] Video sharp and well-lit
- [ ] Captions accurate
- [ ] Graphics readable and professional

---

## CONTINUOUS IMPROVEMENT

### After 30 Days
- [ ] Review analytics (views, watch time, engagement)
- [ ] Analyze quiz performance (are learners passing?)
- [ ] Read all comments/feedback
- [ ] Identify areas for improvement
- [ ] Note lessons learned for next video

### After 90 Days
- [ ] Assess if content needs updating
- [ ] Consider creating supplementary materials
- [ ] Gather feedback from instructors who completed module
- [ ] Update as needed

### Annual Review
- [ ] Check for outdated content, examples, or policies
- [ ] Re-record segments if significant changes needed
- [ ] Update resources and links
- [ ] Refresh graphics/branding if needed
